# Don't Use Buffer or Typefully

## What it is
The rule: don't use third-party scheduling tools (Buffer, Typefully, Hootsuite, Later, etc.) for LinkedIn or X posting. Use the native scheduling features of LinkedIn and X directly, or post live. Third-party tools introduce: (1) reach penalties (LinkedIn deprioritizes posts via API in some periods), (2) format degradation (third-party preview rendering doesn't match native), (3) subscription cost without proportional ROI.

## Why it matters
Scheduling tools sound efficient but the actual ROI is poor. The Sunday batch-writing session produces 5 posts in 2 hours; scheduling them via LinkedIn's native scheduler takes 2 minutes per post (10 min total). Adding a $20–50/month tool to save 10 min/week is a bad trade. The bigger issue is reach: third-party-scheduled posts have historically (and intermittently) been deprioritized by the LinkedIn algorithm.

## How to apply
- **Use LinkedIn's native scheduler.** Built into the post composer; free; no algorithm penalty.
- **Use X's native scheduler.** Premium subscription unlocks scheduling; aligned with X's algorithm.
- **For X without Premium:** post live in the 15 min/day block. The cadence is lower-frequency than LinkedIn anyway.
- **Refuse scheduling-tool subscriptions** — they're a budget leak with low ROI. The subscription dollars buy nothing the native tools don't.

## Examples
- A peer recommends Typefully for X scheduling. Riché checks: $15/month, ~3 hours saved across the year. Decision: declined; native X scheduling (Premium) does the same job and is already paid.
- A Sunday session: Riché writes all 5 LinkedIn posts. Schedules each via native LinkedIn scheduler in ~10 min total. Done.
- An older Buffer account from a previous year is still subscribed at $15/month. Riché cancels — saved $180/year for zero reach loss.

## Anti-patterns
- Adopting scheduling tools "to be more strategic." The strategy is the batch-writing session, not the tool.
- Stacking multiple scheduling tools (Buffer for X, Typefully for threads, Hypefury). Each adds subscription cost; native tools handle all of it.
