# Growth Loops Adapted to Personal Brand (Brian Balfour)

## What it is
Brian Balfour's "growth loops" framework — popularized in his 2018 essay series at Reforge — argues that sustainable growth comes from compounding loops, not one-shot funnels. Each output becomes an input to the same loop, producing exponential rather than linear growth. Adapted to personal brand:

> Content (LinkedIn post, X thread) → peers reshare → audience grows + new peers discover → more peers comment substantively → more inputs to next content → cycle compounds.

The key shift from AARRR: Balfour's loops describe a *self-reinforcing cycle*, not a one-time funnel. Each cycle should leave the system in a stronger state.

## Why it matters
Funnels assume a linear path from acquisition to retention. Loops assume that retained users *become* the acquisition channel for the next cohort. For Riché, this is exactly how the flywheel works: peers who reshare drive new peer discovery, who become resharers themselves. Without loop thinking, you optimize each funnel stage independently and miss the compounding mechanism.

## How to apply
- **Identify the loop:** Riché's primary loop is content → peer reshare → audience growth → new peers in audience → inputs for next content (their questions, frames, pushback) → next content. Each cycle leaves the audience and the inputs richer.
- **Audit the loop's compounding rate.** If the loop runs 1x/week and doesn't grow the resharer pool, it's flat (not compounding). If new resharers join each cycle, it's compounding.
- **Diagnose loop breakage.** A reshare without audience growth = audience isn't expanding. A audience expanding without input richness = the audience isn't engaging substantively. Each break needs a different fix.
- **Cite the canonical source:** Brian Balfour, "Why Product Market Fit Isn't Enough" series (Reforge / brianbalfour.com, 2018+).
- **Pair with the cold-start frame** for the seed phase — see `corpus/growth/metrics/cold-start-personal-brand.md`.

## Examples
- Q1 2026 loop audit: content output stable; reshares per content piece growing (loop is compounding); audience-input richness growing (peers DMing with substantive frames Riché incorporates into next content). Loop healthy.
- Q2 2026 loop audit: content output stable; reshares flat for 6 weeks; audience-input richness stagnant. Loop is broken at the reshare → audience-growth step. Fix: address Segment 2 relationship gap (the reshares aren't reaching new resharers).
- The Helm Labs $3.25M pipeline credibility marker started as a sentence in one Wednesday deep dive. It got reshared, became a frame other peers used, came back as a question Riché answered in a podcast, became a longer story in the next deep dive. Loop output → loop input.

## Anti-patterns
- Treating the funnel as one-way. Loops require feeding outputs back into inputs; without that, growth is linear at best.
- Optimizing one loop step in isolation. The compounding comes from the full cycle running consistently.
